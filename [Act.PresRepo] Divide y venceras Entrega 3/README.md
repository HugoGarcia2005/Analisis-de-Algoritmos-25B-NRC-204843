Primeramente, ejecute el código main.py para acceder al programa.

Ya en el programa se le presentará una interfaz de usuario, escriba o selecciona una ruta donde se encuentre la imagen que desea comprimir.
Seleccione un valor de compresión N (el recomendado es 8) y pulse el botón “Iniciar”.
En la interfaz vera la grafica de complejidad temporal asi como un gif del resultado, en la carpeta donde se encuentra su archivo main.py se creará una carpeta que contendrá las imágenes utilizadas para el gif asi como el mismo gif.

Si quiere cargar una nueva imagen es importante pulsar el botón “Limpiar” y despues escribir o seleccionar la ruta para su nueva imagen (el proceso es el mismo).
